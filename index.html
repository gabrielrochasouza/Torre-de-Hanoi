Torre de Hanoi

A torre é formada pelos discos empilhados no bastão chamado de Start. O objetivo do jogo é transportar a torre para o bastão End, usando a intermediária Offset.

Escolha a dificuldade e depois clique em start para começar.

  • Movimentar uma só peça (disco) de cada vez.
  • Uma peça maior não pode ficar acima de uma menor.
  • Não é permitido movimentar uma peça que esteja abaixo de outra.

Selecione a dificuldade:

  • Fácil
  • Médio
  • Difícil
